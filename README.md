# Vendor Performance Analysis Dashboard

## Project Overview
This project analyzes vendor performance, profitability, and procurement efficiency using SQL, Python, Excel, and Power BI.

## Tools Used
- SQL
- Python (Pandas)
- Excel
- Power BI

## Key Objectives
- Evaluate vendor profitability
- Identify top-performing vendors
- Analyze procurement efficiency
- Generate business insights

## Process
1. Data Cleaning using SQL and Python
2. Data Transformation
3. KPI Development
4. Dashboard Creation in Power BI

## Key Insights
- Identified high-profit vendors
- Detected procurement inefficiencies
- Improved visibility into vendor performance metrics
